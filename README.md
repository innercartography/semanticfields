# Semantic Field

A spatial memory tool for collective meaning-making.

Load a 3D point cloud. Converse with an AI on a Solari flip-board. Group words by color, draw edges between ideas, and place them as annotated nodes in space — building a shared memory palace one session at a time.

**→ [Launch the app](https://innercartography.github.io/semanticfields/semantic-splat.html)**

**→ [Read the docs](https://innercartography.github.io/semanticfields/)**

---

Single HTML file. No build step. No server.  
Requires a modern browser + Anthropic API key (or run inside Claude.ai).
